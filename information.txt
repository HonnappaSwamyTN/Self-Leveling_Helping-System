Kotlin (official Android language)
├── Room Database (local storage - no cloud needed)
├── UsageStatsManager (for app monitoring)
├── WorkManager (for background tasks)
├── AlarmManager (for blocking/reset schedules)
└── Material Design (for UI)

DATA FLOW :
User creates account → Local only (no server)
Set goals/app limits → Stored in Room DB
App usage tracked → UsageStatsManager
Limit exceeded → Blocking service triggered
Daily reset → Automatic at midnight
Inbuilt alaram
